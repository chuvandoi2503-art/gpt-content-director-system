# 1. MỤC ĐÍCH

File này lưu:

* Việc đang thực hiện.
* Việc cần hoàn thành trong giai đoạn hiện tại.
* Trạng thái vận hành hiện hành.

File này được phép thay đổi thường xuyên.

Đây là file làm việc chính của GPT Content OS.
